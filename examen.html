Pregunta 2 Agregar una nueva variable que tenga por nombre average, esta variable debe ser el promedio de bill_length_mm y bill_depth_mm.

df$bill_depth_mm[is.na(df$bill_depth_mm)] <- 0
df$bill_length_mm[is.na(df$bill_length_mm)] <- 0
df <- df %>% 
  mutate(average = ((bill_length_mm+bill_depth_mm)/2))

Pregunta 3 Completar la cantidad de pinginos por species y island, de modo que le permita completar la siguiente tabla (en la hoja).

df %>% 
  group_by(species) %>% 
  summarise(Biscoe = sum(island=="Biscoe"),
            Dream = sum(island=="Dream"),
            Torgersen = sum(island=="Torgersen"))

Pregunta 4 Calcular el valor mximo de body_mass_g para los pinginos de la isla Dream que son female.

df %>% 
  group_by(island) %>% 
  filter(sex=="female" & island=="Dream") %>% 
  summarise(maximo= max(body_mass_g, na.rm = TRUE))
